Проект по сложности вычислений.

Тема: построение правильных раскрасок 3-раскрашиваемых графов за полиномиальное время.

Журавлёв Алексей, 396 группа.

Состав:

1) paper.pdf - основная статья с полным описанием алгоритмов и их реализации.
2) paper.tex - исходник статьи в TeX
3) implementation - папка с исходным кодом по проекту. Язык Python3. Версия интерпретатора 3.5.0. В файлах berger_rompel.py и widgerson.py - реализация основных алгоритмов. В файле help_functions.py - описание вспомогательных функций. tests.py - тестовые запуски алгоритма, написанные в Python unittests. Запуск тестов из этой папки выполняется командой "python3 tests.py"
4) Readme.txt - этот файл
